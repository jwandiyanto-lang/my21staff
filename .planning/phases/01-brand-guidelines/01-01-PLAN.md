Create two SVG logo files:

**wordmark-full.svg** (full wordmark: my21staff)
- ViewBox: 200x40 (or appropriate for text)
- Font: Plus Jakarta Sans, weight 800 (ExtraBold)
- "my" in #2D2A26 (dark text)
- "21" in #F7931A (orange)
- "staff" in #2D2A26 (dark text)
- Use tspan elements for color segments

Example structure:
```svg
<svg width="200" height="40" viewBox="0 0 200 40" xmlns="http://www.w3.org/2000/svg">
  <style>
    @import url('https://fonts.googleapis.com/css2?family=Plus+Jakarta+Sans:wght@800');
  </style>
  <text x="0" y="30" font-family="'Plus Jakarta Sans', sans-serif" font-size="28" font-weight="800">
    <tspan fill="#2D2A26">my</tspan>
    <tspan fill="#F7931A">21</tspan>
    <tspan fill="#2D2A26">staff</tspan>
  </text>
</svg>
```

**icon-only.svg** (just "21")
- ViewBox: 40x40 (square for icons/favicons)
- Font: Plus Jakarta Sans, weight 800
- "21" in #F7931A (orange)
- Centered in viewbox

Example structure:
```svg
<svg width="40" height="40" viewBox="0 0 40 40" xmlns="http://www.w3.org/2000/svg">
  <style>
    @import url('https://fonts.googleapis.com/css2?family=Plus+Jakarta+Sans:wght@800');
  </style>
  <text x="50%" y="70%" text-anchor="middle" font-family="'Plus Jakarta Sans', sans-serif" font-size="28" font-weight="800" fill="#F7931A">21</text>
</svg>
```

Note: SVG uses Google Fonts import for consistent rendering. PNG generation in Plan 02 will embed the font.
