  <name>Task 1: Create StickyCTA component</name>
  <files>src/components/landing/sticky-cta.tsx</files>
  <action>
Create mobile sticky CTA that appears after scrolling past hero:

**Behavior (RESEARCH.md pattern):**
- Hidden initially
- Appears after scrolling ~400px (past hero section)
- Fixed to bottom of viewport
- Only visible on mobile (md:hidden)

**Implementation:**
```tsx
'use client'

import { useState, useEffect } from 'react'
import Link from 'next/link'
import { MessageCircle } from 'lucide-react'
import { WHATSAPP_LINK } from '@/lib/landing-constants'

export function StickyCTA() {
  const [show, setShow] = useState(false)

  useEffect(() => {
    const handleScroll = () => {
      setShow(window.scrollY > 400)
    }
    window.addEventListener('scroll', handleScroll, { passive: true })
    return () => window.removeEventListener('scroll', handleScroll)
  }, [])

  if (!show) return null

  return (
    <div className="fixed bottom-0 left-0 right-0 p-4 bg-white/95 backdrop-blur-sm border-t border-gray-200 md:hidden z-50">
      <Link
        href={WHATSAPP_LINK}
        target="_blank"
        rel="noopener"
        className="flex items-center justify-center gap-2 w-full py-4 bg-landing-cta text-white font-bold rounded-full"
      >
        <MessageCircle className="w-5 h-5" />
        Chat on WhatsApp
      </Link>
    </div>
  )
}
```

**Key details:**
- Use 'use client' for scroll listener
- Add `{ passive: true }` to scroll listener for performance
- z-50 to stay above content
- Subtle backdrop blur for polish
