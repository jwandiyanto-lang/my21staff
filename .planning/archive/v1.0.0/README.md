# V1.0.0 Archive

**Status:** Completed and archived on 2026-01-30

**Reason for archiving:** Restructuring for V2.0 with fresh feature approach while keeping infrastructure (Convex, Vercel, Clerk).

---

## What Was V1.0.0?

Initial implementation of my21staff - WhatsApp CRM + AI assistant for Indonesian SMEs.

**Key accomplishments:**
- Multi-tenant architecture with Clerk organizations
- Convex database with real-time sync
- Kapso WhatsApp API integration
- ARI (AI assistant) with configurable personas
- Dashboard, Inbox, Database, Settings pages
- Offline demo mode for local development
- Eagle Overseas Education workspace (being deactivated)

**Tech stack used:**
- Next.js 15 + React 19 + TypeScript
- Convex (backend)
- Clerk (auth + multi-tenant)
- Kapso API (WhatsApp)
- Shadcn/ui + Tailwind CSS
