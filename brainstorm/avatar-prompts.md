## Master Style Guide

Use this style description for ALL avatars:

```
Style: Minimalist black-and-white flat vector illustration portrait in the "Notion avatar" art style.

LINE WORK:
- THICK, UNIFORM black outlines (monoline weight)
- Lines should feel organic, smooth, and confident — not robotic or geometric

FILLS:
- Solid black fills for: hair, beards, eyebrows, dark clothing
- Pure flat white for: skin, face area, lighter elements

COMPOSITION:
- Clean bust/headshot centered on white background
- No shading, no gray tones, no gradients
- High contrast, clean 2D vector graphics
- Square format (1:1 ratio)
```

## Platform-Specific Tips

### Midjourney
Add to end of prompt:
```
--style raw --s 50 --ar 1:1
```

### DALL-E / ChatGPT
Use prompt as-is. If you get gray tones, add:
```
Absolutely no gray tones, only pure black and pure white.
```

### Leonardo.ai
Select "Illustration" preset, disable "PhotoReal"
