# AIDA Framework — my21staff

## AIDA Summary

| Stage | Message |
|-------|---------|
| **Attention** | No system, no growth. |
| **Interest** | my21staff is the system — AI + human team that actually executes. |
| **Desire** | Finally scale. AI handles the routine, humans handle the complex. |
| **Action** | Get the system. Start growing. |

---

## Attention

**Goal:** Stop the scroll. State the core truth.

### Headlines
- "No system, no growth."
- "Leads everywhere. System nowhere."
- "You can't scale chaos."
- "Still screenshotting chats to save leads?"
- "Masih kerja manual? Gak bakal bisa scale."

### Opening Lines
- "No system, no growth. That's the truth most Indonesian SMEs learn too late."
- "Leads come in — but where do they go? Into chaos."
- "Without a system, you're stuck doing everything manually. You can't scale."

### Pain Points
- Multi-channel chaos — leads scattered everywhere
- No central system — can't build processes
- Manual tracking — screenshot, save, forget
- Growth blocked — stuck at a ceiling
- No visibility — can't see the pipeline
